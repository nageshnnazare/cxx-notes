# Depth-first Search (DFS) Traversal

![drawio](../../../_resources/drawio-{_sketch__false}-9)

(Starting from Node 1) \
1 2 5 6 3 7 8 4

|     | Adjacency list |
| :-: | :------------- |
|  1  | {2,3}          |
|  2  | {1,5,6}        |
|  3  | {1,4,7}        |
|  4  | {3,8}          |
|  5  | {2}            |
|  6  | {2}            |
|  7  | {3,8}          |
|  8  | {4,7}          |

```c++
void dfs(int node, std::vector<int> adj[], int vis[],
   std::vector<int> &ls) {
 vis[node] = 1;
 ls.push_back(node);
 for (auto it: adj[node]) {
  if (!vis[it]) {
   dfs(it, adj, vis, ls);
  }
 }
}
```
