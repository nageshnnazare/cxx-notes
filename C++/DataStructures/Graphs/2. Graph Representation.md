## Input

n = nodes, m = edges \
(n, m) = (5, 6)

Edges: From , To\
2 1 \
1 3 \
2 4 \
3 4 \
2 5 \
4 5

## Storing

### 1. Adjacency Matrix

|  0  |  1  |  2  |  3  |  4  |  5  |
| :-: | :-: | :-: | :-: | :-: | :-: |
|  1  |  0  |  1  |  1  |  0  |  0  |
|  2  |  1  |  0  |  0  |  1  |  0  |
|  3  |  1  |  0  |  0  |  1  |  1  |
|  4  |  0  |  1  |  1  |  0  |  1  |
|  5  |  0  |  0  |  1  |  1  |  0  |

Space = O(n \* m) \
Time = O(m)

```c++
int main(){
 int n, m;
 std::cin >> n >> m;
 int adj[n+1][m+1];
 for(int i=0; i<m; i++) {
  int u, v;
  std::cin >> u >> v;
  adj[u][v] = 1;
  adj[v][u] = 1;
 }
 return 0;
}
```

### 2. Adjacency List

| Node | To Node |
| :--: | :------ |
|  1   | {2,3}   |
|  2   | {1,4,5} |
|  3   | {1,4}   |
|  4   | {3,2,5} |
|  5   | {2,4}   |

Space = O(2 \* E)

```c++
int main(){
 int n, m;
 std::cin >> n >> m;
 std::vector<int> adj[n+1];
 for(int i=0; i<m; i++) {
  int u, v;
  std::cin >> u >> v;
  adj[u].push_back(v);
  adj[v].push_back(u);
 }
 return 0;
}
```
